Why do you need this tool?

This tool is meant to streamline the process of making mock JSON data for demos and PoCs. 20,000+ lines of mock data written in JSON are needed for a single PoC, with lots of intertwined references, interdependencies, and places where it is easy to make one typo and break everything.
"Oh, your device has id 2724 but on line 18,234 you wrote 2723? Won't work! Good luck finding that line."
This app is meant to help create that JSON without having to actually search-and-replace, scour the big file, and spend hours fixing small typos.
