Site

The address of the site is where the pin will be placed on the map in the UI to represent that site - therefore it is important that it is accurate.
You type in an address into the 'Address' field and click 'Show on Map'. This does a search using the Google Maps API, displays it on the map below for verification, and populates the latitude and longitude of that address. This way, you don't have to go figure out the lat/long of the address you want to choose for your site.
